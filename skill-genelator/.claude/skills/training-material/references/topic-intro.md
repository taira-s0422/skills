# 生成AI入門・基礎知識

## 目的

生成AIの基本概念と現状を理解する

## 経営層向け重点

- AIの進化と市場動向（市場規模、成長率）
- 主要プレイヤーと競争環境（OpenAI、Anthropic、Google、Microsoft）
- 自社業界への影響と機会
- 投資判断のポイント

## 管理職向け重点

- 生成AIの仕組み（LLM、トランスフォーマー概要）
- できること・できないこと（限界の理解）
- 主要ツール比較（ChatGPT、Claude、Gemini、Copilot）
- 部門での活用可能性

## 必須コンテンツ

### 1. 生成AIとは

- 定義: テキスト、画像、コードなどを生成するAI
- 従来AIとの違い: ルールベース→学習ベース→生成ベース
- 主な種類: LLM、画像生成、音声生成

### 2. 主要サービス比較

| サービス | 提供元 | 特徴 | 適した用途 |
|---------|-------|------|-----------|
| ChatGPT | OpenAI | 汎用性高、プラグイン豊富 | 一般業務、クリエイティブ |
| Claude | Anthropic | 長文処理、安全性重視 | 文書作成、分析 |
| Gemini | Google | マルチモーダル、検索連携 | リサーチ、データ分析 |
| Copilot | Microsoft | Office連携 | 業務効率化 |

### 3. 最新トレンド

- マルチモーダル化（テキスト+画像+音声）
- エージェント化（自律的タスク実行）
- 企業向けセキュリティ強化
- 業界特化型AI

### 4. 業界別活用事例

**製造業**: 設計支援、品質管理文書、マニュアル作成
**金融**: レポート作成、リスク分析、顧客対応
**小売**: 商品説明、マーケティング、需要予測
**IT**: コード生成、ドキュメント、テスト
