# GITEC Proposal Generator

GITEC（株式会社GI）のAIコンサルティング業務用提案書を自動生成するスキル。

## ワークフロー

```
1. ヒアリング → クライアント情報を収集
2. 課題分析 → 業務課題とAI適用領域を特定
3. 提案書生成 → テンプレートに基づき作成
4. ROI計算 → 費用対効果を数値化
5. 研修プラン → 導入後の教育計画を策定
```

## 提案書構成

### 必須セクション

1. **エグゼクティブサマリー** - 1ページ要約
2. **現状分析** - クライアントの課題
3. **AI導入提案** - 具体的なソリューション
4. **ROI分析** - 費用対効果の数値化
5. **導入ステップ** - 段階的な実装計画
6. **AI研修プラン** - 社員教育プログラム
7. **事例紹介** - 類似業種の成功事例
8. **見積り・スケジュール** - 費用と期間

## ROI計算ガイド

### 時間削減効果の算出

```
年間削減時間 = 対象業務時間 × 削減率 × 従事者数 × 年間稼働日
年間削減金額 = 年間削減時間 × 時給単価
```

### 標準削減率（目安）

| AI適用領域 | 削減率 |
|-----------|-------|
| 文書作成・要約 | 40-60% |
| データ入力・整理 | 50-70% |
| メール対応 | 30-50% |
| 情報検索・調査 | 40-60% |
| レポート作成 | 50-70% |

### ROI計算式

```
ROI = (年間削減金額 - 年間導入コスト) / 年間導入コスト × 100
投資回収期間 = 初期導入コスト / 月間削減金額
```

## AI研修プラン

### 4回シリーズ標準構成

| 回 | テーマ | 時間 | 内容 |
|----|-------|------|------|
| 1 | AI基礎理解 | 90分 | 生成AIの仕組み、できること・限界 |
| 2 | プロンプト基礎 | 90分 | 効果的な指示の出し方、テンプレート |
| 3 | 業務活用実践 | 90分 | 自社業務への適用、ハンズオン |
| 4 | 応用・展開 | 90分 | 高度な活用法、社内展開計画 |

### 研修オプション

- **基礎コース**: 1-2回（入門者向け）
- **標準コース**: 4回（推奨）
- **発展コース**: 6回以上（DX推進担当者向け）

## 導入ステップ

### Phase 1: 準備（1-2週間）
- 現状業務ヒアリング
- AI適用領域の特定
- KPI設定

### Phase 2: 試験導入（2-4週間）
- パイロットチーム選定
- AI環境構築
- 初期研修実施

### Phase 3: 本格展開（1-3ヶ月）
- 全社展開
- マニュアル整備
- 効果測定

### Phase 4: 定着・改善（継続）
- 定期フォローアップ
- 活用事例共有
- 追加研修

## 使用例

### 基本的な使用

```
「製造業向けのAI導入提案書を作成してください」
「従業員50名の建設会社向けにROI計算を含めた提案書を作成」
「4回シリーズのAI研修プランを含む提案書を作成」
```

### 詳細指定

```
「以下の条件でGITEC提案書を作成してください：
- 業種: 小売業
- 従業員数: 30名
- 課題: 在庫管理と顧客対応の効率化
- 予算: 月額10万円以内
- 研修希望: 2回コース」
```
