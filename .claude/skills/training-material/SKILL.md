# Training Material Generator

生成AIコンサルタント向け研修・講義資料を生成するスキル。

## ワークフロー

```
1. 要件確認 → ターゲット、トピック、形式、時間
2. 構成決定 → トピック別ガイドを参照
3. 資料生成 → テンプレートに沿って作成
4. 品質確認 → チェックリストで検証
```

## 要件確認

### ターゲット層

| 層 | 特徴 | 重点 |
|---|------|-----|
| 経営層 | 戦略的視点 | ROI、競合動向、ガバナンス |
| 管理職 | チーム運営 | 実践活用、部下展開、効果測定 |

### 形式

| 形式 | 構成（60分） | 構成（90分） |
|-----|------------|------------|
| 座学 | 導入5分→本編40分→Q&A12分→まとめ3分 | 導入7分→本編60分→Q&A20分→まとめ3分 |
| ハンズオン | 導入5分→デモ10分→演習35分→振り返り10分 | 導入7分→デモ15分→演習50分→振り返り18分 |

## 資料構成

### 座学形式

```markdown
# [タイトル]

## 本日のゴール
- ゴール1
- ゴール2

## アジェンダ
| 時間 | 内容 |
|------|------|

## セクション1: [見出し]
> **Key Message**: [最重要メッセージ]

[内容]

## セクション2-3...

## まとめ
- 要点1-3
- 次のアクション

## 参考資料
```

### ハンズオン形式

```markdown
# [タイトル] - ハンズオン

## 事前準備
- PC、ツールアカウント

## 本日のゴール
- できるようになること1-3

## デモンストレーション
[基本操作の実演]

## 演習1-3
### 目的
### 課題
### 手順
### ヒント

## 振り返り
- 気づきの共有

## お持ち帰り
- プロンプトテンプレート
```

## 品質チェック

- [ ] ターゲット層に適した言葉遣い
- [ ] 時間配分が適切
- [ ] 具体例が十分
- [ ] 持ち帰り価値がある
- [ ] 次のアクションが明確

## 使用例

```
「経営層向けに生成AI入門の研修資料を作成」
「管理職向けプロンプト研修（90分、ハンズオン）を作成」
「リスク管理の1時間講義資料を作成」
```
